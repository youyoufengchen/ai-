引擎说明

4D-Humans HMR2：HMR 2.0 逐帧提取 SMPL 参数，转为标准动作 JSON。
需要下载 HMR2 权重（自动下载或手动放入 ~/.cache/4DHumans/）。GPU 推荐。

MoCapAnything V2 (4D)：TripoSG temporal + video2pose2rot 端到端流水线，支持任意物种骨骼。
需 checkpoints/video2pose2rot/ + TripoSG_temporal/ + RMBG-1.4/ 权重，GPU 推荐。

MoCapAnything V1：占位模式，生成挥手测试动画（验证链路用）。
如已接入真实权重，V2 效果更优。

MotionBERT：MediaPipe 2D 检测 + MotionBERT 3D 提升。
人体专用，精度优于 MediaPipe。

MediaPipe：本地 CPU 推理，33个关键点。
安装 pip install mediapipe 后立即可用。
