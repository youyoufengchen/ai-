# 手动安装 Sentence-Transformers 模型指南

## 方法：浏览器下载 + 本地放置

### 第1步：安装 sentence-transformers 包（小，100KB）
```powershell
cd "D:\新建文件夹 (2)"
.\.venv\Scripts\pip.exe install sentence-transformers --no-deps
```

### 第2步：浏览器下载模型文件（300MB）
打开浏览器访问：
```
https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/tree/main
```

下载以下文件到 `D:\新建文件夹 (2)\temp_model\` 目录：
- [1_Pooling](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/tree/main/1_Pooling)
- [config.json](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/config.json)
- [config_sentence_transformers.json](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/config_sentence_transformers.json)
- [model.safetensors](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/model.safetensors) （最大，300MB）
- [modules.json](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/modules.json)
- [sentence_bert_config.json](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/sentence_bert_config.json)
- [special_tokens_map.json](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/special_tokens_map.json)
- [tokenizer_config.json](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/tokenizer_config.json)
- [tokenizer.json](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/tokenizer.json)
- [vocab.txt](https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/vocab.txt)

### 第3步：放置到缓存目录
下载完成后，将文件放到：
```
C:\Users\你的用户名\.cache\torch\sentence_transformers\sentence-transformers_paraphrase-multilingual-MiniLM-L12-v2\
```

### 第4步：启用向量检索
编辑 `d:\新建文件夹 (2)\modules\action_retriever.py` 第95-98行，取消注释：
```python
if _VECTOR_AVAILABLE:
    self._init_vector_index(embedding_model)
```

### 第5步：重启服务器
```powershell
cd "D:\新建文件夹 (2)"
py server.py
```
