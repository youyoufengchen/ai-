## 快速上手

### 第一步：启动服务

```bash
cd "d:\新建文件夹 (2)"
.venv\Scripts\activate
python server.py
```

### 第二步：打开编辑器

浏览器访问：`http://localhost:8000/frontend/studio-editor.html`

### 第三步：创建场景

1. 从左侧**3D环境**拖拽一个环境到画布
2. 从**道具库**拖拽货架到合适位置
3. 选中货架，在右侧绑定商品
4. 调整位置/旋转/缩放

### 第四步：配置交互

1. 点击右侧 **➕ 添加交互行为**
2. 选择触发条件（如：点击货架A）
3. 选择执行动作（如：播放产品介绍视频）
4. 保存

### 第五步：测试

1. 点击顶部 **📱 直播间模拟器**
2. 发送测试弹幕或点击快捷操作
3. 观察NPC响应

### 第六步：发布

1. 点击 **💾 保存**
2. 点击 **🚀 应用到直播**
