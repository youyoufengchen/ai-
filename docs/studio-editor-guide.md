# 🎬 直播间场景编辑器 - 使用指南

## 概述

直播间场景编辑器是一个**可视化3D场景搭建工具**，让你可以：

- 🌍 **搭建3D直播场景** - 拖拽式布置道具、货架、装饰
- 🎥 **编辑运镜效果** - 设置相机路径、自动旋转
- 🎮 **配置交互行为** - 可视化设置点击后的动作
- 📱 **模拟直播间** - 测试各种直播事件的响应

---

## 快速开始

### 1. 启动编辑器

```bash
cd "d:\新建文件夹 (2)"
.venv\Scripts\activate
python server.py
```

访问：
```
http://localhost:8000/frontend/studio-editor.html
```

## 工作流程

### 步骤1：选择场景模板

1. 从**3D环境**拖拽一个基础环境到画布
2. 或在**场景库**中新建场景

### 步骤2：布置道具

1. 从**道具库**拖拽货架到合适位置
2. 拖拽茶桌、大屏等装饰道具
3. 选中道具，在右侧面板调整：
   - 精确位置（X/Y/Z）
   - 旋转角度
   - 大小比例

### 步骤3：绑定商品

1. 选中货架
2. 在右侧面板**绑定商品**下拉框选择SKU
3. 货架会显示商品名称

### 步骤4：配置交互

1. 点击右侧面板 **➕ 添加交互行为**
2. 配置：
   - **触发条件**：点击/悬停/靠近
   - **触发对象**：哪个货架/大屏
   - **执行动作**：播放视频/展示商品/TTS语音/切换场景
   - **动作参数**：视频URL、商品ID等
3. 保存行为

### 步骤5：设置运镜

1. 点击顶部 **运镜模式**
2. 调整相机到起始位置
3. 在时间轴点击添加关键帧
4. 移动相机到结束位置
5. 添加第二个关键帧
6. 点击 **预览运镜** 查看效果
7. 开启 **自动旋转** 设置环绕速度

### 步骤6：测试

1. 点击顶部 **直播间模拟器**
2. 使用快捷操作模拟：
   - 用户进入
   - 弹幕消息
   - 点赞/送礼
   - 下单
3. 观察3D场景和NPC的响应

### 步骤7：发布

1. 点击 **保存** 保存场景配置
2. 点击 **应用到直播** 推送到实际直播间
3. 在直播间预览最终效果

## 高级功能

### 运镜动画编辑

```
运镜模式开启 → 调整相机位置 → 点击时间轴添加关键帧
                                    ↓
                              移动相机到新位置
                                    ↓
                              添加第二个关键帧
                                    ↓
                              预览/调整插值
```

**关键帧插值方式**：
- 线性插值：相机直线移动
- 贝塞尔曲线：平滑曲线移动（高级）

### 交互行为配置示例

**场景1：点击货架展示商品**
```
触发条件: 点击
触发对象: shelf_001
执行动作: 展示商品
动作参数: {"product_id": "tea_001", "show_popup": true}

效果: 点击货架1 → NPC取货动画 → 展示商品详情 → 播放介绍语音
```

**场景2：点击大屏播放视频**
```
触发条件: 点击
触发对象: screen_001
执行动作: 播放视频
动作参数: {"video_url": "/assets/products/tea_001/intro.mp4"}

效果: 点击大屏 → 大屏开始播放产品视频
```

**场景3：定时自动讲解**
```
触发条件: 定时 (每30秒)
触发对象: shelf_002
执行动作: 播放语音
动作参数: {"text": "各位客官，这款凤凰单丛正在促销中！"}

效果: 每30秒自动触发一次促销话术
```

## 最佳实践

### 场景布局建议

```
        [虚拟大屏 - 展示产品视频]
                ↑
    [货架1] ← [NPC主播] → [货架2]
                ↓
            [茶桌/主播台]
```

### 运镜设计原则

1. **开场**：远景 → 环绕 → 近景主播
2. **产品介绍**：平滑移动到对应货架
3. **互动环节**：轻微摇摆增加活力
4. **结尾**：拉远展示整个场景

### 交互设计建议

- **高频交互**（点击货架）：简单直接，取货+展示
- **特效交互**（大礼物）：全屏特效+NPC特殊动作
- **自动交互**（定时）：不干扰，自然插播

## 故障排除

### 3D场景不显示

1. 检查浏览器是否支持WebGL
2. 刷新页面重试
3. 查看控制台错误信息

### 拖拽放置无效

1. 确保拖拽的是道具图标（不是整个区域）
2. 尝试将物体拖放到可见的地面区域
3. 检查浏览器控制台是否有错误

### 保存失败

1. 检查后端服务是否运行
2. 查看控制台网络请求状态
3. 确保场景ID是唯一的

### NPC动作不触发

1. 确认NPC视频文件存在
2. 检查WebSocket连接状态
3. 在控制台测试动作触发
