# HF Mirror 模型下载步骤

## 直接访问模型页面
浏览器打开：
```
https://hf-mirror.com/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2
```

## 下载这些文件（点击文件名右侧的下载图标 ↓）

必须下载的文件列表：
1. **config.json** ← 点击下载
2. **config_sentence_transformers.json** ← 点击下载  
3. **model.safetensors** ← **最大文件（300MB），必须下载**
4. **modules.json** ← 点击下载
5. **sentence_bert_config.json** ← 点击下载
6. **special_tokens_map.json** ← 点击下载
7. **tokenizer_config.json** ← 点击下载
8. **tokenizer.json** ← 点击下载
9. **vocab.txt** ← 点击下载

还有一个文件夹：
10. **1_Pooling/** 文件夹里的所有文件

## 下载后放置位置

创建文件夹：
```
C:\Users\71082\.cache\torch\sentence_transformers\sentence-transformers_paraphrase-multilingual-MiniLM-L12-v2\
```

把下载的所有文件放进去。
